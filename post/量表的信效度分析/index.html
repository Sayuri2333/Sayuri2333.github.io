信效度分析

问卷在正式定稿之前通常需要进行信度检验和效度检验，以作为编制正式问卷的依据。

信度分析

信度是指一个衡量的正确性或者精确性，信度包括稳定性以及一致性。因此信度分析本身与测量的结果是否准确无关，它的作用在于检验测量本身是否稳定。

问卷的信度主要可以分为三个方面：

  • 可靠性。测验信度越高，表示测验结果越可信，但信度除受到测验质量影响外，也受到很多其他受测者因素的影响，故没有一份测验是完全可靠的。
  • 稳定性。稳定性高的量表(问卷)指一群人在不同时空下接受同样的测度时，结果的差异很小。
  • 一致性。一致性高的问卷指同一群人接受性质相同，题型相同，目的相同的问卷测量后，在各测量结果间显示出强烈的正相关。

对于信度的测量主要可以分为对内在信度以及外在信度进行测量这两个方面。外在信度指指在不同时间对同一批受测对象实施重复测量时，评估结果是否具有一致性。其主要使用重测信度法。而内在信度主要考察一组测度项测量的是否是同一构件，测度项间是否具有较高的内在一致性。测量方法主要为克朗巴哈\(\alpha\)系数，剔除克朗巴哈\(\alpha\)系数以及因子分析。

克朗巴哈系数

克朗巴哈\(\alpha\)系数在1951年Cronbacha 提出，为目前社会科学研究最常用的信度。其量一组测度项的信度，若各测度项反应相同的特质，则各题项间具有真实的相关性。若某题项A与其他题项无相关性，则表明题项A不属于该特质，可以将其从量表删除。其步骤为：

  1. 计算各测度题项的相关系数矩阵，计算相关系数的均值

  2. 计算克朗巴哈\(\alpha\)系数 \( \alpha = \frac { k \bar { r } } { 1 + ( k - 1 ) \bar { r } } \)

其中，\(k\)为测度题项数。\(\bar{r}\)为\(k\)个项目相关系数的均值。克朗巴哈\(\alpha\)系数的取值范围为0至1，一般来说，系数值0.4作为可信与不可信的分界线。大于0.4为可信，其中大于0.7为很可信，而小于0.4通常不可信。

剔除克朗巴哈系数

将某个评估项目剔除后的克朗巴哈\(\alpha\)系数。如果剔除后得到的系数值较剔除前的系数值有显著提高，则说明所剔除的评估项目与其他项目的相关性较低，正是由于剔除后，才使得其他项目的总体相关性得以提高。借此判断题项信度。

信度分析原则

  • 通常针对Likert量尺进行信度分析
  • 若测度项并非全部同向，存在反向题，应对其进行反向处理，再进行信度分析。
  • 若量表包含若干子方面，更强调对各个子方面逐个进行信度分析。这种情况下一般先进行构建设效度分析再进行信度检验。
